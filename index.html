Promise 声明

首先，Promise 肯定是一个类，我们用 class 来声明。

  • 由于 new Promise((resolve, reject) => {})，所以传入一个参数(函数)，Promise/A+ 里叫他 executor (奇怪，我找了半天没找到…)，传入就行了。
  • executor 里面有两个参数，一个叫 resolve（成功），一个叫 reject（失败）。
  • 由于 resolve 和 reject 可执行，所以都是函数，我们用 let 声明。
