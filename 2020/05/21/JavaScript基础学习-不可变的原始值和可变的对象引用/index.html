JavaScript的原始值和对象(基本类型和引用类型)

JavaScript的基础类型(原始值)：undefined、null、string、number、boolean、symbol

原始值和对象区别：

  • 原始值是不可改变的，任何方法都不可改变一个原始值。

原始值和对象的比较

原始值的比较是值的比较。
对象的比较并非是值的比较：即使两个对象包含同样的属性及同样的值，也是不相等的。

对象值是引用，对象的比较是引用的比较：当且仅当它们指向同一个地址内存，它们才相等。
