#### 题目

删除排序数组中的重复项

#### 原题链接

#### 解题思路

此题没怎么读懂什么意思，之前有做过，记不起来当时怎么做的了，直接看代码吧

```
var removeDuplicates = function (nums) {
    let number = 0;
    for (let i = 0; i < nums.length; i++) {
        if (nums[i] != nums[i + 1] || i + 1 === nums.length) {
            nums[number++] = nums[i];
        }
    }
    return number;
}
```
